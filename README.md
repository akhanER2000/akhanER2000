## About

Final-year **Computer Engineering (Ingeniero Civil Informático)** student in Valparaíso, Chile, working across **data science, machine learning and applied AI**.

I don't just doubt a model's output — I translate that doubt into architecture. I build models (local and analytical), analyze data, and ground systems in real documentation so they don't hallucinate. I'm a full engineer, comfortable shipping software end to end, but my center of gravity is **data and models**.

What I do best: enter a field I don't yet know, learn it in depth, and ship a finished, deployed system that works inside it — and I know when to kill a project that no longer makes sense.

**Currently:** final-year capstone (AeroFit) · an independent research line in **AI + quantum computing for drug discovery** (literature-review stage).

**Open to** Data Scientist · AI/ML Engineer · Applied Scientist roles — 2026–2027, Chile or abroad.
